A simple and interactive Movie Recommender System built with Python, using Machine Learning techniques and deployed as a web app with Streamlit.
This project analyzes a movie dataset and recommends similar movies based on user selection.
It uses vectorization techniques and similarity metrics to find movies that are closely related.

🚀 Features
🔍 Select a movie and get similar recommendations
🤖 Machine Learning-based similarity (using sklearn)
📊 Data processing with pandas
🌐 Interactive web interface using Streamlit

🛠️ Tech Stack
Python
pandas
scikit-learn
Streamlit


▶️ Run Locally

Clone the repository:

git clone https://github.com/your-username/movie-recommender.git
cd movie-recommender

Install dependencies:

pip install -r requirements.txt

Run the app:

streamlit run app.py
